# Pro Physique Tracker

A mobile-first PWA for tracking workouts, nutrition, and progress.

## Features

- **Dashboard** - Daily overview with macro tracking, water intake, and stats
- **Training** - Push/Pull/Legs workout program with exercise tracking and weight logging
- **Nutrition** - Food logging with macro breakdown (protein, carbs, fat, fiber)
- **Progress** - Weight trends, workout history, and exercise PRs with charts
- **Settings** - Customizable macro targets and profile

## Tech

- Single HTML file, no dependencies
- LocalStorage for data persistence
- Mobile-optimized with safe area support
- Dark theme

## Usage

Open `index.html` in a browser or deploy to any static host.

Add to home screen on iOS/Android for app-like experience.
